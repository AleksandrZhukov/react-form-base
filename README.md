React Form Base
===============

Base Form component for building convinient forms for [Redux](https://github.com/reactjs/redux).

## Installation

```
npm install --save react-form-base
```

## Documentation

## Running Demo

```
> git clone git@github.com:akuzko/react-form-base.git
> cd react-form-base/demo
> npm i
> gulp
```
